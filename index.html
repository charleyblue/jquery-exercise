Manipulating Items

Let's take a look at a list example again.

Let's select the list with $('#exercise2tasks');

We can inject a new item into that list with the append method: $('#exercise2tasks').append('<li>Another item</li>');

Run that a few times and you'll see we have added a few new items.

We can remove items just as easily - let's get rid of all the list items with $('#exercise2tasks li').remove();

The unordered list is still there but it no longer has any list items. Let's put one back in $('#exercise2tasks').append('<li>Another item</li>');

You can run these methods on any elements in the DOM that you can access via their selectors.

Event Handlers

If you click the button that says "click me", you will see that nothing happens.

We can change that with jQuery by defining event handlers that will fire events when a user does something. Common event handlers would be things like clicking an element or changing a value in a form.

Let's select our button with it's id selector $('#exercise3button'); - hit enter and you'll see we've targeted the button.

Now we are going to do a click event on this button and give it a function to use: $('#exercise3button').on('click', function(event) { alert('Click me button clicked'); });

We can remove this event handler just as easily which will stop the button from firing any events: $('#exercise3button').off('click');

Now let's try an on change handler. This will watch the element for any changes, in the case of an text input, it will fire when the value changes. $('#exercise3value').on('change', function(event) {console.log('value changed!')});

Now change the value in the text box and check your JS console for the output. You will have to tab out of the field to fire the event.
